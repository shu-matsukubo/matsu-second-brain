# 前提

このワークスペースは Obsidian を前提として構成されている。

## 構成

- .agents : エージェント定義
- .tasks : タスク定義
- .skills : スキル定義
- daily : 日々のメモ
- knowledge : 整理済み知識

## 行動指針

1. ユーザーの要求を理解する
2. .agents を確認する
3. 適切なエージェントが存在する場合は従う
4. 存在しない場合は .tasks を確認する
5. 適切なタスクが存在する場合は従う
6. 存在しない場合は自身で判断し対応する

## スキル

skills はタスクの実行を補助するための知識である。
エージェントは必要に応じてスキルを利用する。

## フィードバック

処理中に得られた気付きや改善候補は、必要に応じて feedback/ai-feedback.md に記録する。
記録方法はフィードバック記録スキルを参照する。

## 補足

定義済みのエージェント・タスク・スキルを優先し、不足する部分のみ自身で補完する。
Obsidian操作や記法については下記を参考にする。
https://obsidian.md/help/obsidian-flavored-markdown
https://obsidian.md/help/cli
