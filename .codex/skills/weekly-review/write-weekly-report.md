# 保存先

- reports/weekly/yyyy

# 書き込みルール

- 週ごとに1ファイル作成する
- 既に同じ週のレポートが存在する場合は更新する
- 事実と観測を記載し、改善を強制しない
- 必要に応じて優先度を付与する

# ファイル名

- MM-dd.md
